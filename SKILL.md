# Finance Skill — Инструкции для агента

## Когда активировать

Активируй этот скилл, когда пользователь:
- Сообщает о трате или покупке ("потратил", "купил", "заплатил", "оплатил")
- Спрашивает о расходах ("сколько потратил", "покажи расходы", "аналитика")
- Просит сравнить периоды ("сравни январь и февраль")
- Присылает чек или фото чека
- Спрашивает про бюджет или категории расходов
- Хочет увидеть графики или тренды расходов
- Просит экспорт данных

## CLI путь

```
python3 ~/.openclaw/workspace/skills/finance/fin.py <command> [options]
```

## Валюта по умолчанию

- **EUR** — основная валюта для всех расходов
- **GEL** — использовать для локальных трат в Грузии (кафе, транспорт, продукты)
- **EUR** — для всего остального (подписки, онлайн-покупки, переводы)
- Курсы обновляются автоматически при добавлении записи

## Парсинг сообщений пользователя

Примеры маппинга:

| Сообщение пользователя            | Команда CLI                                        |
|------------------------------------|-----------------------------------------------------|
| "потратил 30 евро на обед"         | `fin.py add 30 EUR Еда "обед"`                      |
| "такси 15 лари"                    | `fin.py add 15 GEL Транспорт "такси"`               |
| "оплатил Netflix 15.99"            | `fin.py add 15.99 EUR Подписки "Netflix"`            |
| "купил кроссовки за 89 евро"       | `fin.py add 89 EUR Одежда "кроссовки"`               |
| "аренда 800 евро"                  | `fin.py add 800 EUR Жильё "аренда"`                  |
| "заправил машину 50 лари"          | `fin.py add 50 GEL Транспорт "бензин"`               |
| "сколько потратил в марте"          | `fin.py summary 2026-03`                             |
| "покажи расходы за неделю"          | `fin.py last 20` (и отфильтровать по датам)          |
| "сравни февраль и март"             | `fin.py compare 2026-02 2026-03`                     |
| "график расходов"                   | `fin.py chart pie 2026-03`                            |
| "тренд расходов"                    | `fin.py chart trend`                                  |
| "удали последнюю запись"            | `fin.py last 1` → получить ID → `fin.py delete ID --yes` |

### Удаление записей
- Агент может удалять **только записи, созданные менее часа назад**.
- Если запись старше часа — CLI вернёт ошибку. Агент **НЕ должен** использовать `--force`.
- `--force` — только для ручного использования пользователем напрямую.
- Если пользователь просит удалить старую запись, предупреди его об ограничении и предложи выполнить команду вручную с `--force`.

### Правила парсинга
1. Если валюта не указана явно — использовать **EUR**
2. "евро" / "€" → EUR, "лари" / "₾" → GEL, "доллар" / "$" → USD, "гривна" / "₴" → UAH
3. Определять категорию по контексту (еда, транспорт и т.д.)
4. Если категория неясна — спросить пользователя
5. Дата по умолчанию — сейчас. "вчера" → вычислить дату, передать --dt

## Работа с чеками

Когда пользователь присылает фото чека:
1. Сохранить файл во временную директорию
2. Проанализировать содержимое (OCR / описание от пользователя)
3. Создать запись с `--receipt /path/to/file --source receipt-ocr`
4. Файл автоматически скопируется в `~/.openclaw/workspace/memory/finance/receipts/YYYY-MM/`

## Отправка графиков

Когда пользователь просит аналитику в виде графика:
1. Вызвать `fin.py chart <type> <period>`
2. Команда выведет путь к PNG файлу в stdout
3. Отправить этот файл пользователю в Telegram как изображение

## Примеры полных сценариев

### Добавление расхода с чеком
```bash
python3 ~/.openclaw/workspace/skills/finance/fin.py add 45.50 GEL Еда "ресторан" --receipt /tmp/check.jpg --source telegram
```

### Месячный отчёт
```bash
python3 ~/.openclaw/workspace/skills/finance/fin.py summary 2026-03
python3 ~/.openclaw/workspace/skills/finance/fin.py chart pie 2026-03
# → отправить текстовый отчёт + PNG в Telegram
```

### Сравнение месяцев
```bash
python3 ~/.openclaw/workspace/skills/finance/fin.py compare 2026-02 2026-03
python3 ~/.openclaw/workspace/skills/finance/fin.py chart compare 2026-02 2026-03
```

### Экспорт
```bash
python3 ~/.openclaw/workspace/skills/finance/fin.py export 2026-03 --format csv
# → выведет путь к CSV файлу
```

## Формат ответов пользователю

После добавления записи подтверди в формате:
```
✅ Записал: обед — 30.00 EUR (категория: Еда)
```

Для сводки — отправь текстовый вывод `summary` как есть (он уже отформатирован).

Для графиков — отправь PNG файл с коротким описанием.
